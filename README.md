# genlicense

#### 介绍
使用rsa进行公钥私钥加密解密
生成rsa加密的软件授权码

致谢farmerx/gorsa


#### 安装教程

1.  go get github.com/Jerryhax/genlicense


#### 使用说明

1.  cd github.com/Jerryhax/genlicense
2.  设置公钥私钥（推荐openssl生成）
3.  go run genlicense.go，输出加密结果字符串
